`[Модель устройства] + [Симптом / Проблема] + [Гео-тег Кемерово] + [Интент: ремонт / цена / как понять]`

**Примеры из матрицы 1000:**
1. Замена матрицы MacBook Pro 14 M3 в Кемерово: цена и сроки на Дзержинского.

## 3. Требования к SEO и Разметке (Инструкция для скрипта)
- **Объем**: 3000 - 4500 знаков.
- **H1**: Содержит Модель + Симптом + Кемерово.
- **URL (Slug)**: Транслит (например: `remont-lidara-xiaomi-roborock-s7-kemerovo`).
- **Title**: До 70 символов. (Пример: *Ремонт лидара Xiaomi Roborock S7 в Кемерово | Админ.Ко*).
- **Description**: До 160 символов. Строго с УТП (диагностика 0₽, гарантия 12 мес).
- **Гео-маркеры (LSI)**: Использовать в тексте фразы: "Кемерово", "на Дзержинского, 2Б", "в Центральном районе", "кузбасский".
- **Формат**: HTML-файл с готовой версткой Tailwind для прямой вставки на сайт.

## 4. Расписание выкладки (Cron)
- **Темп**: 15 статей в день.
- **Время**: Пакетная генерация ночью (03:00 MSK).
- **План**: 1000 статей будут сгенерированы и выложены за 66 дней (~2 месяца). Это безопасный темп для Google/Yandex, исключающий пессимизацию за спам.
