# Админ.Ко — Сервисный центр (Кемерово)  
**Профессиональный ремонт цифровой техники с 2014 года**  
🔗 [admin-ko.ru](https://admin-ko.ru)

---

## ✨ Особенности проекта (v3.0)

*   **Дизайн-система**: Современный минимализм (Stitch) с акцентным красным цветом (`#E50914`).
*   **Доверие (E-E-A-T)**: Интеграция реальных отзывов из Яндекс.Карт и интерактивный FAQ.
*   **Картография**: Уникальный хаб карт (Яндекс, 2ГИС, Google) с мгновенным переключением.
*   **Лидогенерация**: Форма обратного звонка через Formspree (AJAX) + быстрые переходы в Telegram и Max Messenger.
*   **Соответствие 152-ФЗ**: Полный пакет документов (Политика, Согласие, Правила) и Cookie-compliance (18+).
*   **Статистика**: 12 лет опыта, 16 сотрудников, 15 000+ успешных ремонтов.

## 📊 Прайс-лист и данные
Прайс-лист подгружается динамически из `price.json`, который можно обновлять вручную или через экспорт из Google Sheets. Система поиска работает на стороне клиента (client-side), что обеспечивает мгновенный отклик.
